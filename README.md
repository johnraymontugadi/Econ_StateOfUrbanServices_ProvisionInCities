# Econ_StateOfUrbanServices_ProvisionInCities
github source code for economics final group project

## About

This project explores the **state of urban services provision in cities** — examining how essential services are delivered, who has access to them, and what gaps remain. Our website presents the group's research, analysis, and findings on the topic.

## Tech Stack

- [Next.js 16](https://nextjs.org) (App Router) + TypeScript
- [Tailwind CSS v4](https://tailwindcss.com) for styling
- [shadcn/ui](https://ui.shadcn.com) for components
- [lucide-react](https://lucide.dev) for icons

## Running Locally

```bash
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000).

Available scripts: `npm run dev` (start dev server), `npm run build` (production build), `npm run start` (run built app), `npm run lint` (lint check).
